### 문제 설명

<p>일반적으로 오름차순 정렬은 $1$번째 원소에 가까워질수록 작은 수를 놓는 것을 말한다. 이와 달리 K-정렬은 $K$번째 원소에 가까워질수록 작은 수를 놓는 것이다. 민찬이와 준이는 K-정렬을 이용한 K-정렬 게임을 하려고 한다. 게임 방법은 다음과 같다.</p>

<p>$1$부터 $N$까지의 서로 다른 정수가 적힌 돌 $N$개가 일렬로 놓여 있고, $K$가 주어진다. 앞에서부터 $i$번째 위치의 돌에 적힌 정수를 $A_i$라고 할 때, 민찬이부터 시작해서 번갈아 가며 다음 행동 중 하나를 수행한다.</p>

<ul>
	<li>$1 \leq i < j \leq K$와 $A_i < A_j$를 만족하는 $i$, $j$에 대해서 $i$번째 돌과 $j$번째 돌의 위치를 바꾼다.</li>
	<li>$K \leq i < j \leq N$과 $A_i > A_j$를 만족하는 $i$, $j$에 대해서 $i$번째 돌과 $j$번째 돌의 위치를 바꾼다.</li>
</ul>

<p>더 이상 이 두 행동 중 어느 것도 할 수 없는 참가자가 지게 된다.</p>

<p>수열 $A$와 정수 $K$가 주어지고 각 참가자는 이기기 위해 최적의 선택을 한다고 할 때, 이기는 참가자를 출력하자.</p>

### 입력 

 <p>첫 번째 줄에 두 정수 $N$, $K$가 공백으로 구분되어 주어진다.</p>

<p>두 번째 줄에 $N$개의 정수 $A_1, A_2, \cdots, A_N$이 공백으로 구분되어 주어진다.</p>

### 출력 

 <p>민찬이가 이긴다면 <span style="color:#e74c3c;"><code>Minchan</code></span>을, 준이가 이긴다면 <span style="color:#e74c3c;"><code>Junee</code></span>를 출력한다.</p>
