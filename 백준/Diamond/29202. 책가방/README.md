### 문제 설명

<p>오늘도 여느 때와 같이 KSA 학생들은 견우관(기숙사) 1층을 나와서 형설관(강의실)으로 가는 오르막을 걷고 있다. 월요일 1교시는 공강이기 때문에 공부할 책 $K$권을 기숙사에서 가지고 나와야 한다. 기숙사 방에는 총 $N$권의 책이 있는데, $i$번 책은 무게 $W_i$와 부피 $V_i$, 그리고 두께 $T_i$를 가진다.</p>

<p>누구든지 무겁고 큰 책가방을 메고 오르막을 걷는 것은 싫어하기 때문에 <strong>피로도</strong>를 최소화하려고 한다. <strong>피로도</strong>는 책가방에 들어있는 $K$권의 책들의 무게를 모두 더한 값을 $S$, 부피 중 최댓값을 $M$, 그리고 두께 중 최솟값을 $m$이라고 할 때, $S + M + m$로 정의한다. 이미 공부하느라 머리를 충분히 써버린 KSA 학생들을 위해 책가방에 어떠한 책들을 넣어야 하는지 알려주자!</p>

### 입력 

 <p>첫 번째 줄에 두 정수 $N$과 $K$가 공백으로 구분되어 주어진다.</p>

<p>$i + 1$번째 줄에 세 정수 $W_i$, $V_i$, $T_i$가 공백으로 구분되어 주어진다. $(1 \le i \le N)$</p>

### 출력 

 <p>첫 번째 줄에 <strong>피로도</strong>의 최솟값을 출력한다.</p>

<p>두 번째 줄에 책가방에 넣어야 하는 책들의 번호를 나타내는 $K$개의 정수를 공백으로 구분하여 출력한다.</p>

<p>정답이 여러 개 존재한다면 그중 아무거나 출력해도 상관없으며, 각 번호를 출력하는 순서는 상관없다.</p>
