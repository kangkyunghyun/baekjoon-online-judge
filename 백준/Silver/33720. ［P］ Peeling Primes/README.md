### 문제 설명

<p>양의 정수 $N$이 주어질 때, 아래 연산을 몇 번 반복해야 $0$이 되는지 구해보자.</p>

<ul>
	<li>$N$을 나누는 가장 작은 소수를 $p$라고 할 때, $N$에서 $p$를 뺀다.</li>
</ul>

### 입력 

 <p>첫째 줄에는 양의 정수 $N$이 주어진다. $(2 \le N \le 10^{12})$</p>

### 출력 

 <p>첫째 줄에 $N$에 최소 몇 번의 연산을 적용해야 $0$이 되는지 출력한다.</p>
